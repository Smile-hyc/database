# Frontend

React + Vite + Tailwind CSS frontend for the World Cup player card guessing system.

## Commands

```bash
npm.cmd install
npm.cmd run dev
npm.cmd run build
```

## Pages

- 首页 Dashboard
- 世界杯竞猜页
- 球员卡行情页
- 积分排行榜页
- 管理员赛果录入页
- 比赛管理页

The visual style is inspired by modern World Cup event pages and lightweight admin dashboards. No third-party copyrighted page code or player images are copied.
